E-commerce adaptive site for the furniture company with products coming from the external API. App is built with React using:

-   styled components
-   react hooks
-   react router
-   Authentication implemented using Auth0 React
-   Payments using stripe
-   Axios

Pages

-   Home Page
-   Products Page
-   Single Product Page
-   About Page
-   Error Page
-   Cart Page
-   Checkout Page
